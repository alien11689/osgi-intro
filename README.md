# osgi-intro

## Building

```
mvn clean install
```

## Running container

```
./runKaraf.sh
```

## Adding new tasks

```
./addTask.sh taskName description123
```

## Listing tasks

As JSON:

```
./listTasksJson.sh
```

As XML:

```
./listTasksXml.sh
```

## Opening karaf management console

Via script on ubuntu

```
./openMgmConsole.sh
```

or open browser: http://localhost:8181/system/console

## Opening task UI

Via script on ubuntu

```
./openUI.sh
```

or open browser: http://localhost:8181/ui
